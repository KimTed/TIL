  * unzip
    * 특정 폴더에 풀기
      ```
      unzip [압축파일].zip -d [폴더명]
      ```
    * 압축 파일에 포함된 파일 정보 보기 (소문자 L)
      ```
      unzip -l [압축파일].zip 
      ```
    * 압축파일의 오류 여부 확인
      ```
      unzip -t [압축파일].zip
      ```
